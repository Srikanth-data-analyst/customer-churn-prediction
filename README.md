# Customer Churn Prediction (Classification)

## Overview
Predict whether a customer is likely to churn using supervised machine learning. Includes EDA-ready data, feature prep, model training, and evaluation.

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib

## How to run
```bash
pip install -r requirements.txt
python src/train.py
```

## Dataset
A small sample dataset is included: `data/churn_sample.csv`.
